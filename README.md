# 🌐 Online Job Portal

A simple **Online Job Portal Website** built using **HTML, CSS, and JavaScript**.
This platform allows candidates to search for jobs and companies to manage job postings.

---

## 🚀 Features

* 👤 User Signup & Login
* 🧑‍💼 Candidate Dashboard
* 🏢 Company Registration & Login
* 🔍 Job Search Page
* 📄 Company Profile
* 📞 Contact Page
